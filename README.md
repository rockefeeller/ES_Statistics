# 🇪🇸 Sostenibilidad de las Pensiones en España (Proyección 2026)

Este proyecto presenta un **mapa interactivo** que analiza la salud del sistema público de pensiones en España por Comunidades Autónomas, basado en proyecciones demográficas y de afiliación para el año 2026.

## 📊 ¿Qué analiza este mapa?

El objetivo es visualizar de forma clara y directa la **Tasa de Dependencia** y, sobre todo, el **Ratio de Trabajadores por Pensionista**. 

En un sistema de reparto como el nuestro, este número es vital: nos indica cuántas personas activas "sostienen" económicamente a cada jubilado en las distintas regiones.

### 🔍 Hallazgos Clave (Datos 2026)
* **Zonas en Alerta:** Regiones como el Principado de Asturias o Galicia presentan ratios cercanos a **1.6 trabajadores por pensionista**, lo que plantea un reto demográfico importante.
* **Zonas Estables:** Madrid, Canarias y Baleares mantienen ratios por encima de **3.0**, mostrando una pirámide poblacional más equilibrada.
* **El "Semáforo":** El color verde indica mayor sostenibilidad, mientras que el rojo señala las áreas con mayor presión sobre el sistema.

## 🛠️ Características del Mapa
- **100% Interactivo:** Pasa el ratón por cada comunidad para ver los datos detallados.
- **Glosario Integrado:** Incluye definiciones claras para entender la metodología.
- **Diseño Responsivo:** Adaptado para visualizarse correctamente tanto en escritorio como en dispositivos móviles.

## 📚 Fuentes de Datos
Los datos utilizados para este análisis provienen de fuentes oficiales y proyecciones estadísticas:
- **INE (Instituto Nacional de Estadística):** Proyecciones de población por comunidad autónoma.
- **Seguridad Social:** Datos de afiliación y pensiones.
- **Análisis Propio:** Basado en la evolución de las tasas de dependencia históricas.
